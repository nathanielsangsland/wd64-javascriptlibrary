First Day Notes:

Notes

  • When using Emmet, class is noted by a "." and id is by #
  • HTML stands for Hypertext Markup Language
  • DOCTYPE states the type of our document, In this case,HTML
  • our html tag denotes our language to be in English per the lang="en"
  • UTF-8 specifies character encoding. Each character (letter, number, even spaces) hsa a numeric value.
  • viewport notes the device we are reading our document on
  • Title relects the name and displays on the tab within our browser
  • A Styke tag can help us inject css into our document. *Not the best practice.
  • Links allow us to connect to other files, such as our CSS (stylesheet)
  • The Body tag houses all the other tags that we want to display onto our brower. This is what the user sees.
